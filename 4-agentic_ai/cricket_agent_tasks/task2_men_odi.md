# Task 2: Cricket Agent for Men ODI Matches

## Objective
Extend the cricket agent web app to support men's ODI (One Day International) matches worldwide. The app should:
- Fetch and display live odds for men's ODI cricket matches
- Aggregate relevant ODI cricket news
- Provide AI-powered analysis using cricket-specific personas
- Allow user to inspect raw data and analysis prompt

## Requirements
- Build on the structure of `cricket_agent.py` (Task 1)
- Add support for men's ODI matches (do not include Test or T20 in this task)
- UI should allow selection of ODI matches, show odds, news, and analysis
- Use the same cricket-specific analyst personas
- All code and config for this task should be reversible by removing this folder

## Deliverables
- Updated `cricket_agent.py` (now supports men ODI and T20)
- This requirements file
- Any supporting files needed for this task
